# 🛒 UnidSells - Online Marketplace

##  Description

UnidSells is a full-stack web application where users can buy and sell products, chat with other users, and manage their listings.

---

## Features

* User Signup & Login
* Add / Edit / Delete Products
* Real-time Chat System
* Wishlist Feature
* Admin Panel

---

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB

---

##  Project Structure

* `models` → Database schemas
* `routes` → API routes
* `public` → Frontend files
* `uploads` → Product images

---

##  How to Run Project

1. Clone the repository
2. Install dependencies:

   ```
   npm install
   ```
3. Start the server:

   ```
   node server.js
   ```
4. Open in browser:

   ```
   http://localhost:3000
   ```
